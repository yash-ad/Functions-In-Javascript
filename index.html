Functions-In-Javascript

A javascript function is a block of code designed to perform a particular task.

Function declaration:-

    Syntax Of a Function looks something like below :-

  1. function myFunc(){ //This a without parameter example. };

  2. function logicFunc(parameter1,parameter2) { //Here the parameters behave as local variables //Here is a function with parameter } logicFunc(x,y); //Function Invocation

Function declaration is known as (Build) moreover Function invocation is known as (call).

Function invocation is a way to use the code inside the function.

In javascript you can call (Invoke) a function by typing the function name followed by the open and close parantheses.

Parameters are sort of like variable names in addition to the values passed to the function are called Arguments.

A function can also Return a value , the value is returned back to the caller

Function Expression :-
